# 庭木の病害虫一覧表

## 1. 害虫一覧

| 害虫名 | 具体的な種類（代表例） | 予防・対策 |
| :--- | :--- | :--- |
| **アブラムシ** | ワタアブラムシ、モモアカアブラムシ | **予防**: 風通しを良くする、窒素肥料の過多を避ける。<br>**対策**: 見つけ次第捕殺、粘着テープ、薬剤散布（オルトラン等）、牛乳スプレー。 |
| **カイガラムシ** | ルビーロウムシ、イセリアカイガラムシ、ツノロウムシ | **予防**: 風通しを良くする、冬期にマシン油乳剤を散布。<br>**対策**: 成虫は殻が硬いため歯ブラシ等でこすり落とす。幼虫期に薬剤散布。 |
| **グンバイムシ** | ツツジグンバイ、ナシグンバイ | **予防**: 乾燥を避ける（葉裏に水をかける）。<br>**対策**: 葉裏に寄生するため、葉裏に向けて薬剤散布。 |
| **カメムシ** | チャバネアオカメムシ、クサギカメムシ、ツヤアオカメムシ | **予防**: 除草を行う（隠れ家を減らす）。<br>**対策**: 見つけ次第捕殺。スミチオン等の薬剤散布。 |
| **ハダニ** | ミカンハダニ、カンザワハダニ | **予防**: 乾燥を好むため、葉水（はみず）を与える。<br>**対策**: 殺ダニ剤の散布（抵抗性がつきやすいため薬剤ローテーションが必要）。 |
| **ハマキムシ** | チャハマキ、リンゴコカクモンハマキ | **予防**: 葉が巻かれているのを見つけたら早めに摘み取る。<br>**対策**: 巻かれた葉の中にいる幼虫を捕殺、薬剤散布。 |
| **シンクイムシ** | モモシンクイガ、ナシヒメシンクイ | **予防**: 袋かけを行う。<br>**対策**: 被害果実を摘果・処分。孵化直後に薬剤散布。 |
| **コガネムシ** | マメコガネ、ドウガネブイブイ、アオドウガネ | **予防**: 防虫ネットの使用。<br>**対策**: 成虫は捕殺（朝方は動きが鈍い）。幼虫（根を食害）には土壌へのダイアジノン等の薬剤混和。 |
| **カミキリムシ**<br>（幼虫：**テッポウムシ**） | ゴマダラカミキリ、キボシカミキリ | **予防**: 幹に産卵防止のネットや塗布剤を使用。<br>**対策**: 成虫は捕殺。幼虫は排泄物（木くず）が出ている穴に専用ノズルで殺虫剤を注入、または針金で刺殺。 |
| **イラガ** | ヒロヘリアオイラガ、イラガ | **予防**: 冬の間に繭（まゆ）を見つけて除去。<br>**対策**: 幼虫（毒棘あり）に注意して葉ごと除去・捕殺。薬剤散布。 |
| **アメリカシロヒトリ** | アメリカシロヒトリ | **予防**: 早期発見（巣網の中に群生している時期）。<br>**対策**: 巣網ごと枝葉を切り取り焼却・踏み潰す。拡散後は薬剤散布。 |
| **マイマイガ** | マイマイガ | **予防**: 冬の間に卵塊（壁や幹など）を剥ぎ落とす。<br>**対策**: 若齢幼虫期に薬剤散布。 |
| **チャドクガ** | チャドクガ | **予防**: 早期発見（葉裏に整列している時期）。<br>**対策**: 毒毛針に注意し、直接触れずに枝ごと切除して処分。固着剤で固めてから処理すると安全。 |

## 2. 病気一覧

| 病気名 | 原因・媒介昆虫 | 予防・対策 |
| :--- | :--- | :--- |
| **うどんこ病** | **原因**: 糸状菌（カビ）。<br>**環境**: 乾燥気味で昼夜の温度差がある時期に発生しやすい。 | **予防**: 風通しと日当たりを良くする。窒素肥料を控える。<br>**対策**: 重曹スプレー、初期に殺菌剤散布、病葉の焼却。 |
| **黒星病** | **原因**: 糸状菌（カビ）。<br>**環境**: 雨が多い時期、低温多湿で発生。 | **予防**: 落ち葉（越冬源）の処分。マルチングで泥はねを防ぐ。<br>**対策**: 発病した葉の除去、殺菌剤散布。 |
| **赤星病** | **原因**: 糸状菌（カビ）。<br>**環境**: ビャクシン類（中間宿主）が近くにあると発生。 | **予防**: ナシ等の近くにカイヅカイブキ等を植えない。<br>**対策**: 発病葉の摘み取り、殺菌剤散布。 |
| **炭疽病** | **原因**: 糸状菌（カビ）。<br>**環境**: 高温多湿、水はけが悪いと発生しやすい。 | **予防**: 水はけと風通しを改善。密集を避ける。<br>**対策**: 発病部位の切除・焼却、薬剤散布。 |
| **灰色かび病** | **原因**: 糸状菌（カビ）。<br>**環境**: 低温多湿、特に花弁や枯れ葉に発生しやすい。 | **予防**: 咲き終わった花がらや枯れ葉をこまめに除去。多湿を避ける。<br>**対策**: 殺菌剤散布。 |
| **すす病** | **原因**: すす病菌（カビ）。<br>**媒介**: アブラムシ、カイガラムシ等の排泄物に寄生。 | **予防**: アブラムシ・カイガラムシ等の害虫を防除する。<br>**対策**: 原因となる害虫を駆除し、黒くなった部分は拭き取るか洗い流す。 |
| **もち病** | **原因**: 糸状菌（カビ）。<br>**環境**: 梅雨や秋雨の多湿時期、ツツジ類やツバキ類に多い。 | **予防**: 風通しと日当たりを確保。<br>**対策**: 「餅」のように膨らんだ病葉を早期に見つけ摘み取り焼却。 |
| **胴枯病** | **原因**: 糸状菌（カビ）。<br>**環境**: 樹木の傷口から侵入。 | **予防**: 剪定後の切り口に癒合剤を塗布。枝枯れを防ぐ。<br>**対策**: 被害部を削り取り、薬剤塗布。ひどい場合は伐採。 |
| **白紋羽病** | **原因**: 糸状菌（カビ）。<br>**環境**: 土壌中の病原菌が根に感染。 | **予防**: 有機物の未熟堆肥を避ける。排水改良。<br>**対策**: 感染した根の切除、土壌消毒、温湯処理。治療困難な場合が多い。 |
| **根頭がん腫病** | **原因**: 細菌（バクテリア）。<br>**環境**: 土壌感染、接ぎ木や根の傷口から侵入。 | **予防**: 健全な苗木の購入。連作を避ける。<br>**対策**: こぶ（癌腫）を切除し、木酢液や薬剤による傷口消毒。器具の消毒徹底。 |
| **モザイク病** | **原因**: ウイルス。<br>**媒介**: アブラムシ等の吸汁性害虫、ハサミ等の器具。 | **予防**: アブラムシ等の防除。使用器具の消毒。<br>**対策**: **治療法なし**。感染株は抜き取って焼却処分し、蔓延を防ぐ。 |
