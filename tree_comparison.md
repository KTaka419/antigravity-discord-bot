# 似ている庭木・品種違いの見分け方ガイド

このガイドでは、見た目が似ていて区別がつきにくい庭木の見分け方をまとめました。
特に、**葉の付き方（対生・互生）** や **冬場の特徴（樹皮・冬芽）** など、花や実がない時期でも見分けられるポイントを重視しています。

---

## 1. キャラ vs イチイ
どちらもイチイ科の常緑針葉樹ですが、庭木としての扱いが異なります。
※ここでは「キャラ」＝キャラボク（*Taxus cuspidata* var. *nana*）として解説します。

| 特徴 | キャラボク（伽羅木） | イチイ（一位） |
| :--- | :--- | :--- |
| **全体樹形** | 根元から多くの枝が出て株立ち状になりやすい。横に広がる。 | 主幹が立ち上がり、高木になる（直立性）。 |
| **葉の並び方** | **螺旋（らせん）状**に不規則につく。 | 枝に対して**2列に綺麗に並ぶ**（ハネ型）。 |
| **用途** | 玉散らし、生垣、和風庭園の低木。 | 生垣、シンボルツリー、高木。 |

> [!TIP]
> **見分けの決定打**: 枝の先端を見て、葉が四方八方に散らばっていれば「キャラ」、平面的に左右に並んでいれば「イチイ」です。

## 2. ヤマボウシ vs ハナミズキ vs サンシュユ
ミズキ科の落葉樹御三家。花（総苞片）と樹皮で見分けます。

| 特徴 | ハナミズキ（アメリカヤマボウシ） | ヤマボウシ（山法師） | サンシュユ（山茱萸） |
| :--- | :--- | :--- | :--- |
| **開花時期** | 4月〜5月（葉が出る前〜同時） | 5月〜7月（葉が出た後） | **早春（2〜3月）** |
| **花（総苞）の形** | 先端が**くぼんでいる**（M字型）。色は白・ピンク・赤。 | 先端が**尖っている**。色は白（稀にピンク）。 | 小さな黄色い花が集まって咲く（総苞片はない）。 |
| **果実** | 秋に赤い楕円形の実（バラバラにつく）。 | 秋に赤く丸い実（サッカーボール状）。食べられる。 | 秋に赤い長楕円形の実（グミに似る）。 |
| **冬芽（花芽）** | **丸くて平たい**（ボタンのような形）。独特。 | 尖った円錐形。 | 丸くて小さいものが集まってつく。 |
| **樹皮（成木）** | **ワニ皮のように細かくひび割れる**。 | 比較的なめらかで、古くなると薄く剥がれる（鹿の子模様）。 | 薄く剥がれて毛羽立つような質感。 |

> [!TIP]
> **冬の見分け方**: 枝先に「ボタンのような丸いツボミ」があればハナミズキ。「ワニ皮の樹皮」もハナミズキの大きな特徴です。サンシュユは早春に黄色い花で一面を染めます。

## 3. ツゲ vs イヌツゲ
名前は似ていますが、全く別の科です。これが分かれば脱初心者。

| 特徴 | ツゲ（本ツゲ・黄楊） | イヌツゲ（犬黄楊） |
| :--- | :--- | :--- |
| **科名** | ツゲ科 | モチノキ科 |
| **葉の付き方** | **対生**（茎の同じ位置から両側に葉が出る）。 | **互生**（茎の互い違いの位置から葉が出る）。 |
| **葉の形** | 全縁（ギザギザがない）。先端が少し凹むことがある。 | 浅い鋸歯（ギザギザ）がある場合が多い。少し反る。 |
| **実** | フクロウのような形の乾いた実。 | 黒い小さな球形の実。 |

> [!TIP]
> **見分けの決定打**: 枝を見て、葉が「対生（向かい合わせ）」なら本ツゲ。「互生（互い違い）」ならイヌツゲです。庭木として玉散らしに使われるのはイヌツゲが多いです。

---

## 4. ツツジ類の見分け方
ツツジ科ツツジ属は種類が多いですが、庭木でよく見るものの分類です。

| 種類 | 特徴・見分け方 |
| :--- | :--- |
| **ヤマツツジ** | **半落葉性**。朱色の花が代表的。葉は大きめで春葉と夏葉がある。おしべは5本。 |
| **キリシマツツジ** | 常緑（寒冷地では落葉）。花が小ぶりで非常に多花性。深紅色の花が有名。おしべは5本。 |
| **ヒラドツツジ** | **大型**。公園や街路樹の定番。葉も花もこれらの中では最大級。おしべは10本程度あることが多い。 |
| **サツキツツジ** | **開花が遅い**（5月後半〜6月、「皐月」）。葉が小さく硬めで光沢がある。低く刈り込める。 |
| **アザレア** | 西洋ツツジ。花が豪華で八重咲きなどが多い。寒さにやや弱い（鉢植えが多い）。 |
| **ミツバツツジ** | **落葉性**。早春に紫色の花が咲く。開花後、**枝先に3枚の葉**が輪生状に出るのが名前の由来。 |

## 5. モミジ類（カエデ類）
葉の切れ込みと品種による違いです。
※ここでは庭木用語としての分類を重視します。

| 種類 | 特徴 |
| :--- | :--- |
| **イロハモミジ** | 最も一般的。葉は小さめで**5〜7裂**。葉の縁のギザギザ（鋸歯）が重鋸歯で不揃い。 |
| **ヤマモミジ** | イロハより葉が大きく、7〜9裂。鋸歯は揃っている。主に日本海側に自生。 |
| **オオモミジ** | 葉が大きく、鋸歯が細かく整っている（細かいギザギザ）。 |
| **ノムラモミジ** | 「野村」。**春の新芽の時期から赤い葉**が出る品種（夏場は緑褐色になり、秋にまた赤くなる）。 |
| **シダレモミジ** | 枝が垂れる品種群。葉が細かく裂けている（レース状）「アオシダレ」「ベニシダレ」が多い。 |
| **カツラ** | ※品種名としての「カツラ（桂）」。葉が淡い黄緑色（黄金色）で芽吹き、縁が赤オレンジ色に染まる美しい品種。 |

> [!NOTE]
> 樹木としての「カツラ（カツラ科）」は全く別物です（丸いハート型の葉、甘い香りがする）。

## 6. ツバキ vs サザンカ
ツバキ科の似たもの同士。

| 特徴 | ツバキ（椿） | サザンカ（山茶花） |
| :--- | :--- | :--- |
| **開花時期** | 冬〜春（12月〜4月）※早咲きもあり | 晩秋〜冬（10月〜12月） |
| **花の散り方** | **花首からポトリと落ちる**。 | **花弁がバラバラと散る**。 |
| **花形** | 筒状に咲くものが多い（平開しない）。 | 平たく開いて咲く。 |
| **葉の特徴** | 葉柄（軸）に毛がない。葉裏の主脈も無毛。 | 葉柄や若枝に**細かい毛が生えている**。 |

> [!TIP]
> **冬の見分け方**: 花がない時期は、その年伸びた新しい枝や葉の軸をよく見てください。「毛が生えていればサザンカ」、「ツルッとしていればツバキ」です。

---

## 7. コニファー類（針葉樹）
種類が膨大ですが、葉の形状で大きく3つに分けられます。

1.  **鱗葉（りんよう）タイプ**: 
    - ヒノキ、カイヅカイブキ、コノテガシワなど。
    - 葉が鱗（うろこ）のように平たく重なっている。手触りが比較的柔らかい。
2.  **針葉（しんよう）タイプ**: 
    - マツ、モミ、トウヒ、一部のジュニパー（ブルーヘブン等）。
    - 葉が針のように尖っている。チクチクする。
3.  **両方持つタイプ**: 
    - ビャクシン類（ジュニパー）は、若木や剪定の刺激で「先祖返り」を起こし、鱗葉の中に鋭い針葉が混じることがある。

> [!TIP]
> **ゴールドクレスト（ヒノキ科）**: 明るい黄緑色で香りが良い（サンショウのような香り）。根が浅く倒れやすい。

## 8. トネリコ vs シマトネリコ
庭木で「トネリコ」と呼ばれる場合、9割は「シマトネリコ」です。

| 特徴 | シマトネリコ | トネリコ（本株） |
| :--- | :--- | :--- |
| **常緑/落葉** | **常緑〜半常緑**（関東以北では冬に落葉することも）。 | **完全落葉樹**。 |
| **葉の質** | 小さく艶があり、軽やか。 | シマトネリコより少し大きく、艶が少ない。 |
| **自生地** | 台湾、沖縄などの亜熱帯。寒さに弱い。 | 日本（本州〜九州）の山地。寒さに強い。 |
| **用途** | 洋風ガーデンのシンボルツリー。 | バットの原料（アオダモの親戚）。自然風の庭。 |

---

## 9. 柑橘類（ミカン、キンカン、ユズ、レモン）
葉とトゲで見分けます。

| 種類 | 葉の特徴（翼葉） | トゲ | その他 |
| :--- | :--- | :--- | :--- |
| **ユズ（柚子）** | **翼葉（葉の付け根の小さな葉）が大きい**。 | **非常に鋭く長い**。 | 耐寒性が強い。 |
| **レモン** | 翼葉はないか、ごく小さい。若葉が赤みを帯びる。 | 短いが鋭いトゲがある（トゲなし品種もあり）。 | 寒さに弱い。葉を揉むとレモンの香り。 |
| **ミカン（温州）** | 翼葉は小さい痕跡程度。 | **トゲはない**（これが見分ける最大の特徴）。 | 葉がふっくらしている。 |
| **キンカン** | 葉が小さく細長い。 | 短いトゲがある場合がある。 | 果実が極小。 |

> [!TIP]
> **用語「翼葉（よくよう）」**: 葉柄（茎と葉をつなぐ部分）についているヒレのような小さな葉のこと。ユズはこの翼葉が大きく目立ちます。

---

## 10. バラ科（サクラ、モモ、ウメ等）
「桜切る馬鹿、梅切らぬ馬鹿」と言われるように剪定特性も違います。

| 種類 | 幹・樹皮 | 花弁の形 | 花の付き方 |
| :--- | :--- | :--- | :--- |
| **ウメ** | ゴツゴツしている。黒っぽい。 | 丸い。 | **花柄（軸）がほとんどなく**、枝に直接張り付くように咲く。一芽に一花。 |
| **モモ** | 斑点のような横筋が入る。 | 先が尖る。 | 枝に沿って短く咲く。**一芽に二花**（両側に咲く）ため華やか。 |
| **サクラ** | **横縞（皮目）**が入る。ツヤがある。 | 先端が割れている（ハート型）。 | **花柄が長い**。房状に垂れ下がって咲く。 |
| **サクランボ** | サクラと同様。 | サクラと同様。 | 桜桃（オウトウ）と呼ばれる。花柄が非常に長い。 |

---

## 11. キンモクセイ vs ギンモクセイ
香りと葉の違い。

| 特徴 | キンモクセイ（金木犀） | ギンモクセイ（銀木犀） |
| :--- | :--- | :--- |
| **花の色** | **濃いオレンジ色**（橙黄色）。 | **白〜淡いクリーム色**。 |
| **香り** | 非常に強く、遠くまで漂う。 | キンモクセイより控えめで上品な香り。 |
| **葉の鋸歯** | 葉の縁に**細かいギザギザがある**（全縁のものもある）。 | 葉の縁に**鋭いギザギザ（鋸歯）があることが多い**。 |

※実は個体差があり、葉だけで見分けるのは難しい場合がありますが、一般にギンモクセイの方が葉のギザギザが粗く目立つことが多いと言われます。確実に知るには花期を待ちます。

---

## 12. クロガネモチ vs オウゴンモチ
モチノキ科の常緑樹。葉の縁の色がポイントです。

| 特徴 | クロガネモチ | オウゴンモチ |
| :--- | :--- | :--- |
| **葉の色** | 濃い緑色。葉柄や若い枝が**紫黒色**（これが名前の由来）。 | 葉が若草色で明るい。葉の縁が**黄色く縁取られる**（斑入りのように見える）。 |
| **実** | 赤い実が沢山つく。 | 赤い実がつくが、クロガネモチほど目立たないことも。 |

---

## 13. シャラ vs ヒメシャラ
別名「ナツツバキ」。幹の美しさが魅力。

| 特徴 | シャラ（ナツツバキ） | ヒメシャラ |
| :--- | :--- | :--- |
| **幹の肌** | 灰褐色で、古くなると剥がれて**まだら模様**になる。 | **赤褐色**で非常になめらか。ツルツルしていて美しい。 |
| **葉の大きさ** | 大きい（10cm程度）。鋸歯が粗い。 | 小さい（5cm程度）。全体に繊細な印象。 |
| **花** | 白い花（5-6cm）。 | 白い花（2cm程度）で小さい。 |

---

## 14. トサミズキ vs ヒュウガミズキ
早春に黄色い花を吊り下げる落葉低木。

| 特徴 | トサミズキ（土佐水木） | ヒュウガミズキ（日向水木） |
| :--- | :--- | :--- |
| **花の房** | **長い**。7〜8個の花が連なって垂れ下がる。 | **短い**。2〜3個の花が可愛らしくつく。 |
| **花のおしべ** | 葯（やく）が**暗赤色**で目立つ。 | 葯が**黄色**で、花弁と同色に見える。 |
| **全体** | 枝が太く、野趣あふれる。 | 全体に枝が細く、繊細で優しい雰囲気。 |

---

## 15. ベリー類（低木果樹）
実のなる低木たち。形状の違い。

| 種類 | 樹形・トゲ | 実の特徴 |
| :--- | :--- | :--- |
| **ブルーベリー** | 低木（ブッシュ状）。**トゲなし**。ツツジ科なので花は釣鐘型。 | 先端に王冠のような跡がある青い実。紅葉も美しい。 |
| **ラズベリー** | 地下茎で増える直立性の杖（ケーン）。**細かいトゲがある**。 | 集合果。中が空洞で帽子のような形。赤や黄色。 |
| **ブラックベリー** | つる性または半つる性。**鋭いトゲがある**（トゲなし品種もあり）。 | 集合果。中が詰まっている。黒く熟す。 |
| **ジューンベリー** | **小高木**（3m以上になる）。バラ科。トゲなし。 | 6月（June）に赤い実が黒く熟す。サクラのような白い花が咲く。 |

> [!TIP]
> **ジューンベリーだけは「木」**です。他はブッシュやツルとして扱います。シンボルツリーになるのはジューンベリーだけです。
